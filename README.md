# The ReadMe of blog

此仓库是《青丝成雪记》。根据**信息流的类型**，文章分为读后感、听后感、观后感和玩后感；根据**否定之否定的阶段**，文章分为前尘、拂尘和同尘。

**横向地**读，你可以读到最新版的我；**纵向地**读，你可以读到我的历史。《青丝成雪记》是无差别的外宣——不会考虑读者是谁。如果你读完了所有的文章，请通过[Twitter/Xingzhi_Lau](https://twitter.com/Xingzhi_Lau)和我约会或面基。谢谢。
